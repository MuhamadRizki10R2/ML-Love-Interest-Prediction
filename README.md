📌 About Project
This project is a Machine Learning-based chat analysis system that predicts whether someone is interested in you based on chat conversation patterns.
The model uses the XGBoost Algorithm to classify chat interactions into categories such as:
❤️ Interested
🤍 Neutral
❌ Not Interested
The system analyzes text patterns, response behavior, word frequency, and conversational features from chat data.
